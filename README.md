# URL Shortening API Landing Page

A responsive landing page that integrates with the Bitly API to shorten URLs. Users can:

- Shorten any valid URL  
- View a list of shortened links  
- Copy shortened links to the clipboard  
- See error messages if the input is empty  

## Live Demo

Check out the working project here: [Live Demo](https://autumn-t-t.github.io/URL-Shortener/#)

## Features

- Fully responsive design for desktop and mobile  
- LocalStorage to persist shortened links across page refreshes  
- Error handling for empty inputs  

## Technologies Used

- HTML, CSS, JavaScript  
- Bitly API  
- GitHub Pages for hosting
